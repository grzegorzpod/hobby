Moja oferta:

Potrzebujesz sprzętu lub oprogramowania np. do pracy dyplomowej, chcesz zrealizować swój unikalny pomysł, a potrzebne ci urządzenie lub program => przygotuj trochę gotówki i zgłoś się do mnie! Możesz mi zlecić wykonanie zarówno urządzeń elektronicznych jak i oprogramowania:

Przykładowe urządzenia elektroniczne / systemy

  • Cyfrowe termometry, higrometry, przetworniki innych wielkości fizycznych
  • Przetworniki analogowo/cyfrowe, rejestratory, oscyloskopy, systemy akwizycji danych,
  • Sterowniki włącz / wyłącz i z regulacją mocy A/C i D/C,
  • Programatory i czytniki,
  • Komunikacja po interfejsach: RS232/485, LPT, USB, I2C, 1-WIRE, Bluetooth LE, Ethernet, WIFI,
  • Przetwarzanie numeryczne np. RK4, FFT,
  • Graficzna wizualizacja pomiarów i procesów,
  • Bazy danych SQLite, MySQL, BDE, Firebase, ...
  • Moje urządzenia są proste, energooszczędne, łatwe w użytkowaniu i przyjazne dla środowiska.

Tworzenie oprogramowania

  • Programy desktopowe dla DOS: QBasic, Turbo Pascal
  • Programy desktopowe dla Windows 9X/XP/7/10: Delphi, Visual Basic i C# .NET Framework
  • Programy dla urządzeń z Windows CE: Visual Studio
  • Programy dla urządzeń z mikrokontrolerami ATTiny/ATMega/ESP32: C, Arduino, Bascom AVR, Micropython
  • Programy dla urządzeń z linuxem: C, Bash, Python
  • Moje programy są proste, nie obciążają CPU, energooszczędne, stabilne, gotowe do pracy 24/7 i przyjazne dla środowiska.
